# Supa Shots - AI Image Generation for Figma

Powerful Figma plugin to transform a single image into 9 professional shots using AI.

## Features

- **Standalone Architecture:** No external backend required.
- **Dual Modes:**
  - **Human Portrait:** 9 cinematic angles.
  - **Premium Product:** 9 editorial styles.
- **Auto-Detection:** Automatically identifies selected images on canvas.
- **Grid Layout:** Inserts results into a 3x3 grid with one click.
- **Optimized for Figma:** Built for stability and performance within the Figma environment.

## Usage

1. **Install:** Import `manifest.json` into Figma.
2. **Setup:** Enter your Gemini API key in the settings.
3. **Generate:** Select an image and click **Generate 9 Shots**.
4. **Insert:** Click **Insert All** to add variations to the canvas.

## Development

- **Tech Stack:** React 19, TypeScript, Vite.
- **Build:** `npm run build` from `ui-app` directory.
